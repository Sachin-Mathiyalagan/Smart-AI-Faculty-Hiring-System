## ğŸ§© System Architecture  

**1. Job Posting & Resume Upload**  
HR posts job openings; candidates upload resumes.  

**2. Data Preprocessing & Extraction**  
AI extracts skills, qualifications, and experience using NLP-based parsing.  

**3. Resume-Job Matching (EncoreWebSM Model)**  
Performs semantic keyword matching between job descriptions and resumes.  

**4. Candidate Scoring & Ranking**  
Assigns a match score to each candidate using ML models.  

**5. Result Visualization & Shortlisting**  
Displays ranked candidates on an interactive HR dashboard.  

**6. Secure Storage & Compliance**  
All candidate data is stored securely following GDPR guidelines.
