# 🎓 SEESA: Smart AI Faculty Hiring System  

## 🧠 Overview  
**SEESA (Smart & Efficient Educational Staff Analyzer)** is an **AI-powered faculty recruitment system** that automates and enhances the academic hiring process.  
The system leverages **Artificial Intelligence (AI)**, **Machine Learning (ML)**, and **Natural Language Processing (NLP)** to streamline resume screening, candidate ranking, and bias-free hiring decisions.  

Traditional academic hiring is often **time-consuming, biased, and manually intensive**. SEESA addresses these challenges by introducing a **fair, efficient, and data-driven approach** that ensures educational institutions select the **most qualified faculty** quickly and transparently.  

---

## 🚀 Key Features  
✅ **AI-Driven Resume Screening** – Automatically extracts skills, qualifications, and experience from resumes.  
✅ **Smart Candidate Ranking** – Uses semantic matching (EncoreWebSM Model) to rank candidates by job relevance.  
✅ **Bias-Free Selection** – Converts gender-specific pronouns to neutral terms (he/she → they/them) for unbiased evaluation.  
✅ **Multi-Format Resume Support** – Accepts resumes in PDF, DOCX, and TXT formats.  
✅ **Automated Notifications** – Notifies shortlisted candidates automatically.  
✅ **GDPR-Compliant Secure Storage** – Ensures privacy and data protection.  

---

## 🧩 System Architecture  

**1. Job Posting & Resume Upload**  
HR posts job openings; candidates upload resumes.  

**2. Data Preprocessing & Extraction**  
AI extracts skills, qualifications, and experience using NLP-based parsing.  

**3. Resume-Job Matching (EncoreWebSM Model)**  
Performs semantic keyword matching between job descriptions and resumes.  

**4. Candidate Scoring & Ranking**  
Assigns a match score to each candidate using ML models.  

**5. Result Visualization & Shortlisting**  
Displays ranked candidates on an interactive HR dashboard.  

**6. Secure Storage & Compliance**  
All candidate data is stored securely following GDPR guidelines.

```bash
npm run dev
```
